### Hexlet tests and linter status:

This repository hosts a CLI utility designed to detect differences between two files. It supports JSON and YAML as input data formats and offers three output formats: stylish, plain text, and JSON.

### Minimum Requirements:

1. Operating System - Linux or MacOS
2. Command Line Interface (CLI)
3. Node.js version 19.1.0

### Installation

1. **Clone the repository:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
git clone git@github.com/alexeenkoivan/frontend-project-46.git

2. **Change the directory:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
cd frontend-project-46

3. **Install dependencies:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
make install

4. **Make the script executable:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
chmod +x bin/gendiff.js

5. **Publish the package locally:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
make publish

6. **Link the package globally:** ![Copy Command](https://img.shields.io/badge/Copy-Command-brightgreen?logo=github)
npm link
